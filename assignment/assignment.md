Here are well-structured assignments you can give your students. They start simple and gradually combine concepts so students actually *apply* what they learn instead of just memorizing.

---

## 🧩 Assignment 1: HTML Basics (Structure & Syntax)

**Focus:**
Self-closing elements, empty spaces, attributes, class, id

**Task:**
Create a simple profile page.

**Requirements:**

* Use at least **3 self-closing elements** (e.g. `<img>`, `<br>`, `<hr>`)
* Demonstrate **empty spaces** using:

  * `&nbsp;`
  * line breaks (`<br>`)
* Add:

  * `id` to one main section
  * `class` to at least 3 elements
* Include attributes like:

  * `src`, `alt`, `href`, `title`

**Example sections:**

* Profile image
* Name
* Bio
* Link to social media

---

## 🎨 Assignment 2: Styling Fundamentals

**Focus:**
`<style>` element, inline styles, classnames

**Task:**
Style your profile page from Assignment 1.

**Requirements:**

* Use a `<style>` tag inside `<head>`
* Apply styles using:

  * **class selectors**
  * **id selectors**
* Use **inline style** on at least 2 elements
* Create reusable classnames like:

  * `.card`
  * `.text-center`
  * `.highlight`

**Goal:**
Students should see the difference between:

* inline styles vs internal styles

---

## 🎯 Assignment 3: Pseudo Selectors

**Focus:**
`:hover`, `:first-child`, `:last-child`

**Task:**
Enhance interactivity.

**Requirements:**

* Add a button or link and:

  * Change color on `:hover`
* Style:

  * First paragraph differently using `:first-child`
  * Last item in a list using `:last-child`

**Bonus:**

* Add `cursor: pointer` on clickable elements

---

## 📦 Assignment 4: CSS Box Model Deep Dive

**Focus:**
Padding, border, margin, width, height

**Task:**
Create a card layout.

**Requirements:**

* Build **3 cards** (like product cards or blog posts)
* Each card must have:

  * `padding`
  * `border`
  * `margin`
  * fixed `width` and `height`

**Goal:**
Students should visually understand spacing differences.

---

## 📐 Assignment 5: Box Sizing

**Focus:**
`box-sizing: border-box` vs `content-box`

**Task:**
Compare both models.

**Requirements:**

* Create **2 identical boxes**
* Apply:

  * `box-sizing: content-box` to one
  * `box-sizing: border-box` to the other
* Add:

  * padding and border
* Observe and explain the difference

**Write-up:**
Students must explain:

> Why the sizes appear different

---

## 🧱 Assignment 6: Block vs Inline Elements

**Focus:**
Display behavior

**Task:**
Demonstrate layout differences.

**Requirements:**

* Use:

  * block elements (`<div>`, `<p>`)
  * inline elements (`<span>`, `<a>`)
* Convert:

  * a block element into inline using CSS
  * an inline element into block

**Goal:**
Understand how layout flow changes.

---

## 🖱️ Assignment 7: Cursor & UX

**Focus:**
`cursor` property

**Task:**
Improve user interaction.

**Requirements:**

* Apply different cursor styles:

  * `pointer`
  * `not-allowed`
  * `text`
* Create:

  * a disabled button (with `not-allowed`)
  * a clickable card (`pointer`)

---

## 🧪 Final Project: Mini Web Page

**Combine EVERYTHING**

**Task:**
Build a small webpage (e.g. “My Favorite Movies” or “Portfolio Page”)

**Must include:**

* Self-closing tags
* Attributes, class, id
* `<style>` + inline styles
* Pseudo-selectors
* Box model (padding, border, margin)
* Width & height
* `box-sizing`
* Block vs inline usage
* Cursor styling

**Bonus ideas:**

* Add hover animations
* Create a grid of cards

---

## 🧠 Teaching Tip

Don’t let them just submit code—ask:

* “Why did you use class instead of id here?”
* “What changed when you switched box-sizing?”
* “Why is margin outside but padding inside?”

That’s where real understanding happens.
