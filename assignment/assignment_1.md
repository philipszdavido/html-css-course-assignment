Assignment 1: HTML Basics

Focus: Self-closing elements, empty spaces, attributes, class, id

Task: Create a simple profile page.

Requirements:

- Use at least 3 self-closing elements
- Demonstrate empty spaces using:
  - &nbsp;
  - line breaks (<br>)
- Add:
  - id to one main section
  - class to at least 3 elements
- Include attributes like: \* src, alt, href, title
  Example sections:
- Profile image
- Name
- Bio
- Link to social media
